# Quantitative Trading System (Python - Quantitative Finance, 2025)

A Python-based quantitative trading project that combines multi-strategy signal generation, vectorized backtesting, paper trading simulation, risk analytics, and automated Excel reporting.

This repository is designed as a portfolio-ready quant project for showcasing practical skills in:

- quantitative research
- strategy development
- portfolio backtesting
- risk management
- trading system design
- Python-based reporting workflows

## Overview

The system runs an end-to-end workflow:

1. Load market data for a basket of liquid US equities and ETFs.
2. Generate signals from multiple trading strategies.
3. Backtest each strategy across the full history.
4. Compute risk and performance statistics.
5. Replay a recent window in a paper-trading simulator.
6. Export a multi-sheet Excel report for analysis and presentation.

The current configuration in [`main.py`](/C:/Users/User/Documents/Playground/main.py) uses:

- Universe: `AAPL`, `MSFT`, `GOOGL`, `AMZN`, `TSLA`, `SPY`, `QQQ`, `NVDA`
- Backtest window: `2020-01-01` to `2024-12-31`
- Backtest capital: `$1,000,000`
- Paper trading capital: `$100,000`
- Paper trading replay window: `120` days

## Strategy Suite

The project includes three long-only strategies:

- `MA Crossover`: dual moving average trend-following using a fast/slow crossover
- `Momentum`: cross-sectional momentum that ranks assets by recent performance
- `Mean Reversion`: Bollinger Band-style long entries when price deviates below trend

Strategy implementations live in:

- [`strategies/moving_average.py`](/C:/Users/User/Documents/Playground/strategies/moving_average.py)
- [`strategies/momentum.py`](/C:/Users/User/Documents/Playground/strategies/momentum.py)
- [`strategies/mean_reversion.py`](/C:/Users/User/Documents/Playground/strategies/mean_reversion.py)

## Key Features

- Vectorized backtesting engine for fast historical simulation
- Modular strategy architecture built around a shared base class
- Portfolio risk metrics including Sharpe, Sortino, VaR, CVaR, Calmar, and max drawdown
- Paper trading simulator with commissions, slippage, execution logs, and position tracking
- Automated Excel report generation with multiple analytics sheets
- Clean project structure suitable for GitHub, interviews, or further extension
